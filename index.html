Projects




TownSquare

TownSquare is a comprehensive solution that promotes community engagement in smart cities. The app provides a platform for citizens to help each other and share resources, allowing them to tackle daily challenges together. TownSquare provides smart city dwellers with the tools they need to build a stronger, more connected community.

Tech Stack: Node, Express, Firebase, HTML, CSS.

fybrrStore

A virtual storage space that is visioned to be supported by a community who wants their data to stay safe with themselves. fybrrStore is a distributed storage application that distributes the data in the network with adequate redundancy which keeps users’ files available all the time. The app constitutes of a File Explorer to view the files and folders.

Tech Stack: Node, Express, Web3.Storage, IPFS.

Smart Insti Web App

A social media for students where they are connected in a private social network, follow each other, like and comment others posts, share resources for course work and post about the ongoing events; they can also register complaints which are visible to the admins. Grabbed 2nd position in a three day IIT Bhilai Hackathon.

Tech Stack: Node, Express, MongoDB, HTML, CSS.

fybrrShare

A WebRTC based peer-to-peer file sharing platform for secure sharing. It eliminates the carbon footprint completely unlike generic file sharing applications which burn almost 4kg of CO2 per hour of use. fybrrShare is highly scalable and can transfer files in low latency.

Tech stack: WebRTC, Express, Node, socket.io

fybrrChat

An instant messaging platform based on peer-to-peer networks for texting, sharing images or files. It also implements voice and video conferencing feature in it. It makes the communication secure and faster by eliminating the need of any central server which reduces latency and jitter by manifold.

Tech stack: WebRTC, Express, Node, socket.io

WebVideoChat

A real-time peer-to-peer video conferencing platform. The platform leverages peer-to-peer (P2P) technology, means communication takes place directly between the users, without the need for a server. The application has an ability to provide a high-quality user experience, with minimal latency and packet loss.

Tech stack: WebRTC, Express, Node, socket.io
